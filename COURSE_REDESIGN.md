# 圆锥曲线课程重构方案

## 基于认知科学的设计原则

### 核心理论框架

| 理论 | 原则 | 在本课程中的应用 |
|------|------|----------------|
| **CRA序列** (Concrete→Representational→Abstract) | 先具体→再图形→最后代数 | 每课：先看D3动画理解几何意义 → 再用Desmos探索 → 最后推导公式 |
| **认知负荷理论** (Sweller) | 工作记忆容量有限，需分解复杂任务 | 圆锥曲线解题拆分为独立子技能，逐个训练再组合 |
| **worked example效应** (g=0.48) | 初学者应先研究解题过程，而非自己摸索 | 每个题型先展示完整worked example，然后渐退(fading) |
| **双编码理论** (Paivio) | 视觉+语言双通道编码记忆更强 | 每个公式都配几何动画，方程与图形始终并列 |
| **几何直觉先于代数操作** | 代数是几何理解的形式化延伸 | 先"看见"曲线和性质，再推导方程 |
| **间隔重复** | 分散练习优于集中练习 | 课程中交替不同曲线类型的练习 |
| **productive failure** (Kapur) | 先尝试再讲解，加深理解 | 每课开头设置探索性问题 |
| **具身认知** | 文科生更适合叙事、情境、视觉方式 | 历史故事引入，生活情境连接 |

### 学生六大认知障碍 (研究证据)

1. **定义掌握不完整** — 忽略限制条件（如2a > |F₁F₂|）
2. **曲线类型混淆** — 椭圆/双曲线标准方程和焦轴搞混
3. **数形脱节** — 纯代数处理，不画图、不用几何直觉
4. **计算过载** — 联立、展开、判别式、韦达…认知负荷爆表
5. **隐含条件遗漏** — 忘记检查Δ>0、参数范围、特殊斜率
6. **死记硬背** — 记公式不理解几何意义，变形就不会做

## 课程重构：基于题型和认知顺序

### 模块结构

```
模块一：看见曲线（几何直觉，CRA-Concrete/Representational）
模块二：掌握工具（代数基础，CRA-Abstract）
模块三：小题速解（选择题/填空题核心题型）
模块四：大题攻略（解答题系统方法）
模块五：提分技巧（高级方法，选学）
```

---

### 模块一：看见曲线 (4课) — 几何直觉优先

**认知目标：** 建立圆锥曲线的视觉直觉，让学生在脑中"看见"这些曲线

| 课 | 主题 | D3可视化重点 | 视频源 |
|----|------|-------------|--------|
| 1.1 | **什么是圆锥曲线** — 用平面切圆锥 | 3D锥体切面动画：改变切面角度 → 圆→椭圆→抛物线→双曲线 | 李永乐BV14s |
| 1.2 | **椭圆：两焦点的舞蹈** — 定义、形状与离心率 | 可拖拽双焦点椭圆：|PF₁|+|PF₂|=2a 实时验证；e滑块改变形状 | 一数ch01,ch05 |
| 1.3 | **双曲线：差的几何** — 定义与渐近线 | 可拖拽双曲线：||PF₁|-|PF₂||=2a；渐近线如何"引导"曲线 | 一数ch04 |
| 1.4 | **抛物线：反射的魔法** — 定义、准线与反射性质 | 光线从焦点出发→反射平行于轴；卫星天线为什么是抛物面 | 一数ch09 |

**设计特点：**
- 无公式推导，纯视觉探索
- 每课以真实生活问题开场（productive failure）
- D3动画为核心教学工具，而非辅助
- 文科生友好：用故事（开普勒发现行星轨道、阿基米德聚光镜）引入

---

### 模块二：掌握工具 (4课) — 从几何到代数

**认知目标：** 从几何定义推导代数方程，建立数形结合能力

| 课 | 主题 | D3可视化重点 | 视频源 |
|----|------|-------------|--------|
| 2.1 | **从定义到方程** — 三种曲线标准方程的推导 | 动画：几何定义 → 坐标化 → 代数化简 → 方程，每步可逆 | 一数ch03 |
| 2.2 | **焦半径公式** — 从定义出发一步推导 | 焦半径动画：P移动时|PF₁|和|PF₂|实时变化，公式实时验证 | 一数ch01,ch02 |
| 2.3 | **离心率** — 一个数如何决定形状 | e从0到1再到>1的连续变化动画，曲线类型切换 | 一数ch05,ch06 |
| 2.4 | **韦达定理速查** — 解析几何中最重要的工具 | 韦达定理可视化：二次方程的根与系数，几何意义 | 一数ch10 |

**设计特点：**
- 每个公式从几何意义出发推导，不要求死记
- worked example + fading模式
- 同时展示方程和图形（dual coding）

---

### 模块三：小题速解 (5课) — 选择题/填空题核心

**认知目标：** 掌握选择题和填空题中的6大高频题型，目标：看到题→识别题型→30秒出答案

| 课 | 主题 | 高频题型 | 视频源 |
|----|------|---------|--------|
| 3.1 | **离心率计算** — 直接求值与范围 | 题型1: 离心率 | 一数ch05,ch06 |
| 3.2 | **标准方程** — 条件→方程 | 题型2: 标准方程 | 一数ch03 |
| 3.3 | **焦点三角形** — 面积与角度 | 题型3: 焦点三角形 | 一数ch01,赵ch22 |
| 3.4 | **渐近线与几何性质** — 双曲线快速判断 | 题型4+5: 渐近线、几何性质 | 一数ch04,ch08 |
| 3.5 | **二级结论速查** — 最实用的12条结论 | 题型6: 定义法求距离 + 结论应用 | 一数ch02,ch07 |

**设计特点：**
- 每课：3个worked example → 3个faded example → 3个独立练习
- 计时功能：训练30秒/题的速度感
- D3动画让每个结论有"画面感"

### 模块四：大题攻略 (6课) — 解答题系统方法

**认知目标：** 掌握圆锥曲线解答题的标准流程和11大题型

| 课 | 主题 | 题型 | 关键技能 | 视频源 |
|----|------|------|---------|--------|
| 4.1 | **通解流程：设→联→韦→代** | 全局方法 | 建立解题框架 | 一数ch11 |
| 4.2 | **直线与曲线的交点** | 位置关系+弦长 | 联立方程、判别式、弦长公式 | 一数ch08,ch10 |
| 4.3 | **定点问题** | 定点/定直线 | 参数化→消参→定点 | 一数ch12 |
| 4.4 | **定值问题** | 定值/常数 | "先猜后证"策略 | 一数ch12 |
| 4.5 | **中点弦与点差法** | 中点弦 | 点差法避开韦达 | 一数ch14 |
| 4.6 | **面积、范围与存在性** | 面积/最值/存在 | 韦达+面积/优化 | 一数ch13 |

**设计特点：**
- 子技能分离训练（cognitive load管理）：
  - 第一遍：只练"联立"步骤
  - 第二遍：只练"韦达"步骤
  - 第三遍：组合完整流程
- 每课的D3动画展示解题过程的"动态版"
- 高亮常见错误点（红色警告框）

---

### 模块五：提分技巧 (3课) — 高级方法（选学）

**认知目标：** 为冲刺满分的学生提供简化计算的高级工具

| 课 | 主题 | 视频源 |
|----|------|--------|
| 5.1 | **齐次化与硬解定理** — 简化计算量 | 一数ch15,ch16 |
| 5.2 | **仿射变换** — 椭圆↔圆的快速通道 | 一数ch17,ch18 |
| 5.3 | **极点极线与射影几何** — 高考命题背景 | 一数ch19,ch20,ch21 |

**设计特点：**
- 标注为"选学"，不影响核心学习路径
- 每种技巧只在它能显著简化计算时使用
- 对比：用常规方法 vs 用技巧，计算量差异可视化

## 每课标准结构（基于认知科学）

```
┌─────────────────────────────────────────────┐
│ 1. 探索问题 (Productive Failure, 2分钟)      │
│    → 一个直觉性问题，先让学生思考             │
├─────────────────────────────────────────────┤
│ 2. 几何直觉 (CRA-Concrete, 5分钟)            │
│    → D3动画展示核心概念的几何意义              │
│    → 可拖拽、可调参数的交互体验               │
├─────────────────────────────────────────────┤
│ 3. 公式推导 (CRA-Abstract, 5分钟)            │
│    → 从几何→代数，每步有动画对应              │
│    → 方程与图形并列显示（dual coding）         │
├─────────────────────────────────────────────┤
│ 4. Worked Example (认知负荷管理, 5分钟)       │
│    → 完整解题过程，子步骤高亮标注             │
│    → D3逐步展示解题流程                      │
├─────────────────────────────────────────────┤
│ 5. Faded Example (渐退练习, 3分钟)           │
│    → 部分步骤留空，学生填入                   │
├─────────────────────────────────────────────┤
│ 6. Desmos探索 (交互验证, 3分钟)              │
│    → 自由调参，验证公式，建立直觉             │
├─────────────────────────────────────────────┤
│ 7. 参考视频 (Bilibili嵌入)                   │
│    → 一数/赵礼显的相关讲解片段               │
├─────────────────────────────────────────────┤
│ 8. 速查表 + 常见错误 (元认知)                │
│    → 关键公式 + ⚠️ 易错点红框                │
└─────────────────────────────────────────────┘
```

## 与现有课程的关键区别

| 维度 | 传统教学 / B站视频 | 本课程 |
|------|-------------------|-------|
| 内容顺序 | 按教材/视频发布顺序 | 按认知难度梯度：几何→代数→应用 |
| 首要媒介 | 板书/PPT讲解 | D3交互动画 + Desmos探索 |
| 公式教学 | 给公式→背→做题 | 几何意义→推导→验证→应用 |
| 解题教学 | 看老师做→自己做 | 子技能分离训练→worked example→fading |
| 文科生适应 | 不区分对待 | 叙事引入、具身体验、视觉优先 |
| 错误处理 | 事后纠错 | 预先标注高频错误点 |
| 复习机制 | 集中刷题 | 间隔重复 + 交叉练习 |
